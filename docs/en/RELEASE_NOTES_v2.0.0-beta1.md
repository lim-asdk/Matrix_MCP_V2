## Matrix MCP v2.0.0-beta1

**Release Date**: 2026-03-25  
**Release Type**: Public Beta Premiere

First public beta release of Matrix MCP.

### Included
- Desktop and web workspace runtime
- MCP server connection via stdio and SSE
- AI profile system using OpenAI-compatible APIs
- Persona-based prompt loading
- Example server configuration for public testing

### Recommended start
Use Web mode first:

```bash
python run_web_server.py
```

### Notes
- Desktop mode requires WebView2 on Windows
- Local-first setup
- Packaging and broader distribution will come later
