# Matrix MCP v2.0.0-beta1

**Release Date**: 2026-03-25  
**Release Type**: Public Beta Premiere

Matrix MCP의 첫 번째 공식 공개 베타(v2.0.0-beta1) 릴리스를 발표하게 되어 기쁩니다. 이 버전은 기존 내부 엔진을 기반으로 외부 개발자들이 즉시 사용하고 테스트할 수 있도록 정제된 버전입니다.

## 🌟 주요 특징

### 1. Dual Mode Execution
- **Web Browser Mode**: Flask 기반 웹 서버를 통해 모든 브라우저에서 AI 작업 환경에 접속할 수 있습니다.
- **Desktop Native Mode**: Windows 사용자들을 위해 `pywebview` 기반의 단독 윈도우 모드를 제공합니다.

### 2. Full MCP Support
- **stdio Transport**: 로컬 Python/Node.js MCP 서버를 직접 실행하고 도구를 호출할 수 있습니다.
- **SSE Transport**: 원격 엔드포인트를 통해 대규모 도구 인벤토리를 통합할 수 있습니다. (공식 샘플 SSE 포함)

### 3. AI Persona & Profiles
- `.txt` 파일 기반의 페르소나 시스템을 통해 AI의 행동 양식을 자유롭게 정의할 수 있습니다.
- 다중 AI 설정 프로필을 지원하여 다양한 모델(OpenAI, xAI, DeepSeek 등) 사이를 빠르게 전환할 수 있습니다.

### 4. Zero Sensitive Leakage
- `.gitignore` 및 예시 파일 구조를 통해 개인 API Key와 히스토리가 절대 유출되지 않도록 설계되었습니다.

## 🚀 시작하기
1. 저장소를 클론합니다.
2. `requirements.txt`를 통해 의존성을 설치합니다.
3. `.env` 파일에 API 키를 설정합니다.
4. `run_web_server.py`를 실행하여 브라우저에서 AI를 만나보세요!
